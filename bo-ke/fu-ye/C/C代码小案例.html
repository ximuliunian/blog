判断输入数是否为素数

能够被1和本身整除的数就是素数

思路

  1. 先定义,进行输入
  2. 输入的值送到fun函数里面
  3. fun函数的作用是进行判断是否是素数,如果是素数返回正常值的i,如果不是,则返回不正常的i
  4. 在主函数里面用返回值i进行判断,看看是否等于之前输入的,如果等于,则是素数,反之则不是

代码

#include <stdio.h>

//声明函数fun
int fun(int su);

//主函数
int main() {
    int su;
    //输入
    scanf("%d", &su);

    //返回值与数进行比较,若一致,则是素数,否则不是
    if (fun(su) == su) {
        printf("%d是素数", su);
    } else {
        printf("%d不是素数", su);
    }
}

//函数fun,进行对素数的判断
int fun(int su) {
    int i;
    //进行循环,从2开始,结束是输入的值
    for (i = 2; i < su; i++) {
        //判断数除以i等于是否0,如果有,则提前跳出循环,判定不是素数
        if (su % i == 0) {
            break;
        }
    }
    //返回循环后i的值
    return i;
}

计算器

利用 if 或者 switch 分别实现计算器

思路

    利用if实现计算器

  1. 进行定义变量(进行运算的两个数和运算符号)
  2. 实现输入 格式是: 前数 运算符号 后数
  3. 进行if判断,如输入的是 * 则第一步对第一个if进行判断,如果符号相同,则进行输出,如果不是,则继续判断,直至一样,如没有一样的符号,则报出"符号输入错误"

    利用switch实现计算器

  1. 进行定义变量(进行运算的两个数和运算符号)
  2. 实现输入 格式是: 前数 运算符号 后数
  3. 进行switch查找,输入一个符号进行对比,如有一样符号,则输出,如没有一样的符号,则报出""符号输入错误

代码1(if)

#include <stdio.h>

int main() {
    //定义变量
    int qian_shu, hou_shu;
    char fu;

    //输入部分输入格式为 前数 符号 后数
    printf("请输入");
    scanf("%d %c %d", &qian_shu, &fu, &hou_shu);

    //进行判断,输入符号为 + 时进行加法运算
    if (fu == '+') {
        printf("%d+%d=%d", qian_shu, hou_shu, qian_shu + hou_shu);
        //进行判断,输入符号为 - 时进行减法运算
    } else if (fu == '-') {
        printf("%d-%d=%d", qian_shu, hou_shu, qian_shu - hou_shu);
        //进行判断,输入符号为 * 时进行乘法运算
    } else if (fu == '*') {
        printf("%d*%d=%d", qian_shu, hou_shu, qian_shu * hou_shu);
        //进行判断,输入符号为 / 时进行除法运算带余数
    } else if (fu == '/') {
        printf("%d/%d=%d...%d", qian_shu, hou_shu, qian_shu / hou_shu, qian_shu % hou_shu);
    } else {
        printf("符号输入错误");
    }
}

代码2(switch)

#include <stdio.h>

int main() {
    //定义变量
    int qian_shu, hou_shu;
    char fu;

    //输入部分输入格式为 前数 符号 后数
    printf("请输入");
    scanf("%d %c %d", &qian_shu, &fu, &hou_shu);

    //进行选择,如输入符号为 + 时进行加法运算
    switch (fu) {
        case '+':
            printf("%d+%d=%d", qian_shu, hou_shu, qian_shu + hou_shu);
            break;
        case '-':
            printf("%d-%d=%d", qian_shu, hou_shu, qian_shu - hou_shu);
            break;
        case '*':
            printf("%d*%d=%d", qian_shu, hou_shu, qian_shu * hou_shu);
            break;
        case '/':
            printf("%d/%d=%d...%d", qian_shu, hou_shu, qian_shu / hou_shu, qian_shu % hou_shu);
            break;
        default:
            printf("符号输入错误");
    }
}

乘法口诀表

利用for进行打印输出乘法口诀表

思路

  1. 定义两个自增数
  2. 第一层循环（行循环）初始值为 1 循环9次（<=9 ），每循环一次初始值+1
  3. 第二层循环（列循环）初始值为 1 循环 i(行循环的初始值) 次，每次循环初始值+1，直至与 i 持平

    4. 行循环初始值是1列循环的初始值也是1所以就是1*1=1，然后行进行自增由1变2，列循环就是1*2=2 2*2=4 依次按照这个完成整个乘法表

  4. 进行输出在里面要有（i*j）是计算结果的
  5. 在第二层外面第一层里面输入一个换行，防止输出结果混乱不堪

代码

#include <stdio.h>

int main() {
    int i, j;

    //进行循环,循环9次
    for (i = 1; i <= 9; i++) {
        //j就是相乘数的前数
        for (j = 1; j <= i; j++) {
            //进行打印输出
            printf("%d*%d=%d ", j, i, i * j);
        }
        //每次执行之后换行
        printf("\n");
    }
}

水仙花

水仙花数又称为自恋数...它的 个位的三次幂+十位的三次幂+百位的三次幂=自身

思路

  1. 定义变量
  2. 输入数字
  3. 取得水仙花的 个位 十位 百位

    4. 个位直接 %10 就可以取得，因为是整形，所以就相当于除10后取它的余数

  4. 先进性位数判断,水仙花数是三位数,看看输入的数字是否满足三位
  5. 满足三位数则进行最后判断,个个位相乘相加看看到最后是否等于输入的数字

代码

#include <stdio.h>

int main() {
    //声明输入的 水仙花数 和 个十百位
    int shui_xian_hua;
    int ge, shi, bai;

    printf("请输入一个三位数");
    scanf("%d", &shui_xian_hua);

    //取得水仙花的个位十位百位
    ge = shui_xian_hua % 10;
    shi = shui_xian_hua / 10 % 10;
    bai = shui_xian_hua / 100 % 10;

    //首先确认输入的数字是不是一个三位数,如果是则正常输出,不是提示错误
    if (shui_xian_hua > 99 && shui_xian_hua < 1000) {
        //进行比较,三个百位相乘+三个十位相乘+三个个位相乘是否等于输入的数字
        if (bai * bai * bai + shi * shi * shi + ge * ge * ge == shui_xian_hua) {
            printf("%d是水仙花数", shui_xian_hua);
        } else {
            printf("%d不是水仙花数", shui_xian_hua);
        }
    } else {
        printf("输入的不是三位数");
    }
}
